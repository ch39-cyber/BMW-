# BMW SmartDrive 3D Cockpit Simulator

A web-based prototype for the BMW SmartDrive cockpit UI shown in the design image.

## Included features

- Vehicle health and system status panel
- Driving assistant toggles: lane assist, adaptive cruise, collision warning, blind spot monitor
- Driver monitoring with attention gauge, eye status, and face detection indicators
- Live speed, gear, drive mode, and roadmap navigation overlay
- Speed limit indicator and route ETA
- Interactive media player with playback controls and progress bar
- Media playlist switching and track navigation
- Vehicle control actions: lock, unlock, lights, trunk, climate, seats, camera, settings
- Bottom navigation bar with active page switching and voice command simulation
- Speed limit warning state and route progress animation
- Unity prototype scaffold available under `Assets/Scripts/CockpitManager.cs`

## Run locally

1. Open `index.html` in your browser.
2. Interact with the simulator controls and track playback.

## Files

- `index.html` — main simulator page
- `styles.css` — cockpit UI styling
- `app.js` — interaction logic and state simulation
